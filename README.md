# 3D Pac-Man — Наклонная платформа

## Структура проекта

```
pacman3d/
├── main.go       игровой цикл, физика, рендеринг
├── config.json   параметры игры — редактируй без перекомпиляции
├── maze.txt      разметка доски — редактируй без перекомпиляции
└── README.md
```

`config.json` и `maze.txt` должны лежать рядом с exe-файлом.

## Запуск

```bash
go mod init pacman3d
go get github.com/hajimehoshi/ebiten/v2
go run .
```

Сборка exe:
```bash
go build -o pacman3d.exe .
```

## Управление

| Клавиша | Действие |
|---|---|
| `←` `→` | Наклонить доску влево / вправо |
| `↑` `↓` | Наклонить доску от себя / на себя |
| `Q` | Вращать камеру влево вокруг доски |
| `E` | Вращать камеру вправо вокруг доски |
| `R` | Рестарт (угол камеры сохраняется) |

## Редактирование maze.txt

Открой `maze.txt` любым текстовым редактором и нарисуй лабиринт:

| Символ | Значение |
|---|---|
| `#` | Стена |
| `.` | Пелет (+10 очков) |
| `o` | Большой пелет (+50 очков) |
| `P` | Старт пакмана (один на карте) |
| `G` | Старт призрака (до 4 штук) |
| ` ` | Пустое пространство |

Правила:
- Все строки могут быть разной длины — ширина берётся по самой длинной
- `P` не должен стоять вплотную к `#` со всех сторон
- `G` ставь в открытых клетках подальше от `P`
- Края лабиринта лучше обносить стенами `#` чтобы пакман не вылетел

## Редактирование config.json

### Физика пакмана

| Параметр | По умолчанию | Описание |
|---|---|---|
| `Gravity` | 700 | Сила гравитации пикс/с². Больше — быстрее разгон при наклоне |
| `Friction` | 0.91 | Множитель скорости за кадр. Ближе к 1.0 — меньше торможения |
| `MaxSpeed` | 380 | Максимальная скорость пакмана пикс/с |
| `MaxTilt` | 0.38 | Максимальный угол наклона доски в радианах (~22°) |
| `TiltSpeed` | 0.016 | Угловое ускорение наклона за кадр |
| `TiltDamp` | 0.86 | Затухание наклона при отпускании кнопки (0..1) |

### Камера

| Параметр | По умолчанию | Описание |
|---|---|---|
| `CamH` | 500 | Высота камеры над доской. Больше — вид сверху, меньше — вид сбоку |
| `CamD` | 340 | Дистанция камеры от центра доски по горизонтали |
| `CamFovD` | 420 | Фокусное расстояние. Больше — меньше искажений, меньше — рыбий глаз |
| `CamYawSpeed` | 0.025 | Скорость вращения камеры по Q/E рад/кадр |

Угол обзора определяется соотношением `CamH / CamD`:
- Больше `CamH` или меньше `CamD` — камера смотрит сверху
- Меньше `CamH` или больше `CamD` — камера смотрит сбоку

### Мир

| Параметр | По умолчанию | Описание |
|---|---|---|
| `CellSize` | 34 | Размер одной клетки лабиринта в единицах мира |
| `WallH` | 22 | Высота стен |
| `PacRadius` | 10 | Радиус пакмана — влияет на коллизию со стенами |
| `GhostSpeed` | 42 | Скорость призраков пикс/с |
| `GhostTimer` | 55 | Кадров между сменой направления призрака. Меньше — агрессивнее |

### Быстрые рецепты

**Пакман медленный:**
`Gravity` ↑, `Friction` ближе к 1.0, `MaxSpeed` ↑

**Управление слишком инертное:**
`TiltSpeed` ↑, `TiltDamp` ↓

**Призраки слишком быстрые:**
`GhostSpeed` ↓, `GhostTimer` ↑

**Вид слишком сверху:**
`CamH` ↓ или `CamD` ↑

**Вид слишком сбоку:**
`CamH` ↑ или `CamD` ↓

**Искажения по краям экрана:**
`CamFovD` ↑
